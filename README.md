# tftp-go

A TFTP server implementation written in Go.

At the moment it implements following RFCs:
- [RFC 1350](https://datatracker.ietf.org/doc/html/rfc1350)
- [RFC 2347](https://datatracker.ietf.org/doc/html/rfc2347)
